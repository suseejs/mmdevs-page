### Initial setup guide

#### 1. Fork

<!-- markdownlint-disable MD059 -->

Fork `mmdevs.org` repository by clicking [here][fork].

#### 2. Create a branch

```sh
git switch -c your_branch
```

#### 3. Local setup

**Install Dependencies :**

```sh
npm install
```

**Install githooks :**

```sh
npm run hooks:install
```
